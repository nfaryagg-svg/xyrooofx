Xyrooo FX API Documentation

Base URL: https://api.xyrooo.fx/v1

Welcome to Xyrooo FX Rest API Premium — a high-performance multi-purpose API platform delivering 200+ endpoints across AI, downloaders, search, tools, fun, and more. All responses are returned in JSON format.

JSON HTTPS Only UTF-8 REST No CORS
🔑 Authentication

Include your API key in every request via the apikey query parameter or the X-API-Key header.

Example Request
GET https://api.xyrooo.fx/v1/ai/chat?q=hello&apikey=YOUR_KEY

// Or via Header:
X-API-Key: YOUR_KEY
